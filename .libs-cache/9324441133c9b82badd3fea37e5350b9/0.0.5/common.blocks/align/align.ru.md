# Align

Выравнивание элементов внутри блока. Блок используется, с помощью примиксовывания к
любому другому блоку.

## Использование блока

``` js
{
    block : 'paragraph',
    mix : [{ block : 'align', mods : { hor : 'center' } }],
    content : 'Default paragraph'
}
```

## Модификаторы блока

### _hor

Выравнивание элементов по горизонтали.

* left
* center
* right
