# Align

Align elements in blocks content. Use block as mix for another blocks.

## Block usage

``` js
{
    block : 'paragraph',
    mix : [{ block : 'align', mods : { hor : 'center' } }],
    content : 'Default paragraph'
}
```

## Modifiers of a block

### _hor

Align element on horizontal line.

* left
* center
* right
