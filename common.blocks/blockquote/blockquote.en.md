# Blockquote

No comments.

## Block usage

``` js
{
    block : 'blockquote',
    content : 'Lorem ipsum dolor sit amet, consectetuer adipiscing elit.'
},
{
    block : 'blockquote',
    source : 'Anton Winogradov',
    content : 'Lorem ipsum dolor sit amet, consectetuer adipiscing elit.'
}
```
