__Блок: Blockquote__

Блок для оформления цитат в тексте.

Использование блока:

``` js
{
    block : 'blockquote',
    content : 'Lorem ipsum dolor sit amet, consectetuer adipiscing elit.'
},
{
    block : 'blockquote',
    source : 'Anton Winogradov',
    content : 'Lorem ipsum dolor sit amet, consectetuer adipiscing elit.'
}
```
