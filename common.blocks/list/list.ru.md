# List

Используется для создания списков.

# Использование блока:

``` js
{
    block : 'list',
    items: [ 'unordered 1', 'unordered 2' ]
}
```

## Вложенных списков:

``` js
{
    block : 'list',
    items : [
        'item 1',
        [
            'item 2',
            {
                block : 'list',
                items : [ 'nested item 1', 'nested item 2' ]
            }
        ]
    ]
}
```

## Модификаторы блока

### _type

- упорядоченный список (list_type_ordered)

``` js
{
    block : 'list',
    mods: { type : 'ordered' },
    items: [ 'ordered item 1', 'ordered item 2' ]
}
```
